- **Kawasaki:** `XXXXX-XXXX` — always specify model variant (e.g., KLX150BF, KLX110L) as part numbers drift across model years.

### Currency Conversion
- Fixed benchmark: `1 THB = 0.123 MYR` (update only if live rate fetched via Python and delta > 5%).
- All MYR prices rendered to **2 decimal places**.
- If THB source price is unavailable, mark as `PRICE_UNVERIFIED_THB`.

4. Listing title or description explicitly states **"Genuine"** or **"Original OEM"**.

## ⚠️ Anti-Hallucination Guardrails

1. **Never** invent a price range. If data is unavailable → `CHECK_LOCAL_MARKET`.
2. **Never** use "Link" as placeholder text. Generate a functional URL or leave the cell blank with flag `URL_PENDING`.
3. **Always** distinguish part type:
   - `[OEM]` — Honda/Kawasaki genuine only.
   - `[HQR]` — High-Quality Replacement from trusted brand (e.g., Showa suspension, NGK plugs, Renthal chains). HQR is only valid for T2-eligible consumables.
   - `[UNVERIFIED]` — Source not yet validated; block from final output until cleared.
4. **Never** use a T3 (non-Mall Shopee) source in a final Job File without explicit human sign-off.
5. **Always** include model year in part queries for Kawasaki KLX — part numbers drift across variants (KLX110, KLX110L, KLX150BF, etc.).
6. If a Honda part is discontinued, flag `DISCONTINUED — SOURCE_ALTERNATIVE` and source nearest OEM-equivalent; do not silently substitute.

## 📦 Procurement Strategy (Vendor Review Findings)

### Honda OEM — Ordering Guidance
- Pre-order fast-wear parts in bulk to amortise international shipping cost: cam chains, oil seals, valve stem seals, air filters.
- Batch shipments quarterly where possible; individual urgent orders attract disproportionate shipping overhead.
- Maintain a `CRF_FAST_WEAR_LIST` of top-10 consumables for standing reorder.

### Kawasaki OEM — Ordering Guidance
- MY distributor network is thinner than Honda; build a `KLX_BUFFER_STOCK` of top-10 fast-wear parts and replenish once per quarter.
- Always confirm part against model year fiche before ordering; KLX110 and KLX150 share some part numbers with different torque specs.
- Cross-reference with KLX service manual PDF for any diagram gaps.

### Shopee (T2 Mall Only) — Ordering Guidance
- Use only for consumables (filters, gaskets, NGK plugs, brake pads, chain sets).
- Batch orders to reduce per-item shipping cost.
- Log seller name, SKU, and order date in Job File for warranty traceability.
- Requalify sellers each quarter — Mall status and ratings change.
